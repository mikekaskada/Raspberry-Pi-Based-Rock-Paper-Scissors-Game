# B. Εκπαίδευση μοντέλου ΤΝ με Teachable Machine
Το [Teachable Machine](https://teachablemachine.withgoogle.com/) είναι ένα εργαλείο που χρησιμοποιείται για τη δημιουργία απλών μοντέλων Μηχανικής Μάθησης χωρίς την ανάγκη προγραμματιστικών δεξιοτήτων. Μπορεί να χρησιμοποιηθεί από μαθητές, εκπαιδευτικούς και ερευνητές για να δημιουργήσουν μοντέλα Τεχνητής Νοημοσύνης που αναγνωρίζουν πρότυπα σε εικόνες, ήχους ή κίνηση. Το Teachable Machine ανήκει στην κατηγορία της **επιβλεπόμενης μάθησης (supervised learning)** στον τομέα της **Μηχανικής Μάθησης (Machine Learning)**. Η επιβλεπόμενη μάθηση περιλαμβάνει την εκπαίδευση ενός μοντέλου χρησιμοποιώντας δεδομένα με ετικέτες για να αναγνωρίζει μοτίβα και να προβλέπει κατηγορίες.

Στην πλατφόρμα Teachable Machine, οι παράμετροι **Epochs** και **Batch Size** παίζουν σημαντικό ρόλο στη διαδικασία εκπαίδευσης του μοντέλου.

- **Epochs**: Μία εποχή είναι ένας πλήρης κύκλος κατά τον οποίο το μοντέλο περνάει από ολόκληρο το σύνολο δεδομένων εκπαίδευσης μία φορά. Αν για παράδειγμα ορίσουμε τις εποχές στις 100, αυτό σημαίνει ότι το μοντέλο θα επεξεργαστεί το σύνολο δεδομένων εκπαίδευσης 100 φορές. Περισσότερες εποχές μπορεί να οδηγήσουν σε καλύτερη ακρίβεια, αλλά και σε μεγαλύτερο χρόνο εκπαίδευσης. Δεδομένου του μεγάλου αριθμού δεδομένων, ο χρήστης μπορεί να ξεκινήσει με 100 εποχές. Αν παρατηρήσει ότι το μοντέλο δεν βελτιώνεται μετά από κάποιες εποχές, μπορεί να μειώσει τον αριθμό των εποχών για να εξοικονομήσει χρόνο.

- **Batch Size**: Το μέγεθος της παρτίδας αναφέρεται στον αριθμό των δειγμάτων εκπαίδευσης που περνάνε μέσω του μοντέλου πριν από την ενημέρωση των παραμέτρων του. Με ένα batch size των 512, το μοντέλο θα ενημερώνει τις παραμέτρους του μετά την επεξεργασία κάθε 512 δειγμάτων. Ένα μεγάλο batch size μπορεί να κάνει την εκπαίδευση πιο αποδοτική, αλλά απαιτεί περισσότερη μνήμη. Ένα batch size των 512 είναι κατάλληλο για την περίπτωση των 8000 εικόνων της κάθε κατηγορίας, καθώς θα επιτρέπει στο μοντέλο να επεξεργάζεται τις εικόνες πιο αποτελεσματικά. Εάν παρατηρήσει προβλήματα μνήμης, ο χρήστης μπορεί να μειώσει το batch size. 

Αυτές οι παράμετροι επιτρέπουν τον έλεγχο της διαδικασίας εκπαίδευσης και τη ρύθμιση της για καλύτερη απόδοση και ακρίβεια του μοντέλου.
### Παρτηρήσεις
1. Η διαδικασία παίρνει πολύ ώρα.
2. Κατά την διάρκεια της εκπσίδευσης **δεν πρέπει** να πάμε σε άλλο παράθυρο ή σε άλλη καρτέλα.
3. Πολλές φορές θα εμφανιστεί μήνυμα ότι **η ιστοσελίδα δεν ανταποκρίνεται. Θα πρέπει να το αγνοήσουμε**.
